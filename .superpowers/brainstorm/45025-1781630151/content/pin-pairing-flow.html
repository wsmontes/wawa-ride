Fluxo de Dados

App (líder)
  │
  ├─ RideEvent(status: .proposed)
  ├─ mesh.start() — BLE advertising
  ├─ Broadcast: AnnouncePayload(groupID, PIN)
  │   (a cada ~4s via BitChat BinaryProtocol v2)
  │
  ├─ (futuro: follower receives PIN, validates)
  ├─ (futuro: leader adds follower to participants)
  │
  └─ Quando todos conectam: RideEvent.status = .active
